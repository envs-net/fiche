envs.net | TermBin


Netcat-based command line pastebin.




USAGE

you can POST a text:
    echo just testing! | nc tb.envs.net 9999
content of a file:
    cat ~/some_file.txt | nc tb.envs.net 9999
or your terminal output:
    ls -la | nc termbin.com 9999

REQUIREMENTS

there is only one thing you need to use this service - netcat.
netcat is available on most platforms, including Windows, Mac OS X and Linux.

if you run a server and like this site, clone it! centralization is bad.
Source here: https://github.com/envs-net/fiche.
you can also support it financially via liberapay.



ALIAS

to make your life easier, you can add alias to your .bash_aliases on Linux
and .bash_profile on Mac OS X. just remember to reset your terminal session after that.
echo 'alias tb="nc tb.envs.net 9999"' >> .bash_aliases

now you can use:
echo less typing now! | tb
